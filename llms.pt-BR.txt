# atomwrite

> Operações atômicas de arquivo para agentes LLM -- um CLI, zero corrupção

atomwrite é um CLI Rust que fornece 22 subcomandos para manipulação de arquivos com garantias de escrita atômica. Toda escrita usa tempfile + fsync + rename. Toda resposta é NDJSON com checksums BLAKE3. Construído para agentes LLM que precisam de operações de arquivo confiáveis e estruturadas.
